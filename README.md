# Creator Content Analytics Dashboard

A creator-facing analytics dashboard built with **Next.js and TypeScript** to explore performance patterns across short-form content and support **data-informed content strategy decisions**.

This project emphasizes **clean frontend architecture, data pipelines, and interpretability**, and is actively being developed as a foundation for future ML-powered content insights.

> 🚧 **Status:** In active development — features, metrics, and insights are continuously evolving.
> 
> 🔗 **Live Demo:** [View on Vercel](https://creator-dashboard-indol.vercel.app/)


---

## 🚀 Features
- Analytics dashboards for examining **engagement, retention, and creative attributes**
- Modular **Next.js App Router** architecture with clear server/client separation
- Reusable data ingestion and transformation pipeline
- Strategy-focused views designed for extensibility and iteration

---

## 🛠 Tech Stack
- **Next.js (App Router)**
- **TypeScript**
- **React**
- **PostCSS / CSS**
- **Local CSV/JSON data ingestion**

---

## 📈 Current Scope
- Exploratory analytics across **400+ short-form videos**
- Feature-level analysis of content performance drivers
- Creator-facing views designed for clarity, interpretability, and iteration

---

## 🧠 Engineering & Product Philosophy
- Separation of concerns between data loading, state management, and UI
- Typed data flows to improve maintainability and reduce runtime errors
- Prioritizes **pre-posting and early-stage analysis** to minimize algorithm-driven noise
- Designed as a **scalable product surface** with clear extension points for ML integration

---

## 🔮 Roadmap
- Integrate ML-based content performance prediction models
- Add advanced filtering and interaction patterns
- Expand to multi-platform content analytics
